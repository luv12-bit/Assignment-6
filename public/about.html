The stuff I used to build this

Building this server helped me understand what happens behind the scenes when I visit a website.

The Node.js HTTP Module

I used the http module which is built into Node.js. It lets me listen for requests from a browser and send back responses. I learned that the response needs a status code (like 200 for OK) and a header telling the browser what kind of file it is receiving.

The File System (FS)

To show these HTML files, I had to use fs.readFile. It reads the file asynchronously, which means the server doesn't have to stop and wait for the file to finish reading before doing other things.

I'm excited to learn more about how to make these servers even better!
