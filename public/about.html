About This Server

This project explores the core fundamentals of web development with Node.js. By building a server from scratch, I've learned how HTTP requests and responses work, how to set headers, and how to read files from the system asynchronously.

Technologies Used

  • Node.js: The runtime environment.
  • HTTP Module: To create the server and handle requests.
  • FS Module: To read HTML and CSS files from the disk.
  • Path Module: To handle file paths cross-platform.
